Olá! Bem vindo ao meu museu artístico!

Sou o Lucas, estudante brasileiro de programação e adorador das artes visuais. Sempre que desejo descansar ou até refletir, não há nada igual para mim como ouvir uma boa música e desenhar sobre algo que gosto; seja isso uma paisagem, uma pessoa, um objeto ou qualquer coisa (mesmo) que me venha na cabeça.

Com o tempo aprendi a melhorar minhas habilidades com desenho a medida que pratiquei durante minhas pausas no cotidiano. E com meus conhecimentos de programação, decidi compartilhar através desse site algumas das baguncinhas que fiz.

Ferramentas que uso
Desenho Analógico

Gosto de utilizar o lápis e papel às vezes quando estou longe do computador. Esse é o método que me imergiu no desenho, e o que considero ter mais habilidade. As vezes quando quero fazer desenhos detalhados, utilizo esse para esboços e desenho virtualmente com as fotos dos esboços no MediBang

MS Paint

Para esboços mais simples, ou com alguns estilos específicos (como pixel art), o Microsoft Paint é muito bom para alcançar esses efeitos. Qualquer máquina Windows de versão anterior ao 10 tem esse programa, e não é difícil de usar.

MediBang Paint

Essa é uma plataforma que utilizo para desenhos com uma mecânica muito parecida com a do Photoshop. Faço desenhos com efeitos e diversos filtros utilizando o MediBang porque além de ser gratuito, é muito rico em ferramentas e possui suporte para armazenamento em nuvem.

Blender

O Blender é uma plataforma gratuita de modelagem 3D que comecei a explorar este mesmo ano. Não é uma plataforma para desenhos 2D, mas é impossível não citá-la quando relacionadas a meu desenhos: Às vezes, utilizo referências nele para ilustrar melhor as posições que quero, e também o utilizo para criar modelos 3D; mesmo que eu seja iniciante ainda.
